Research Topic

Our topic is about crimes around each bike station in Chicago area.

We want to make a web app where you can find how many crimes around any bike station in the map, and you can check the types and counts of crimes!

Data Sources and Data Clean

There are two main data sources:

First one is the divvy data, this gives us geometirc information of all bike stations in the city. For the divvy data, I selected the latest data from Ferb 1st to July 1st to count the stations. From it, I picked three columns "FROM STATION ID","FROM LONGITUDE","FROM LATITUDE", kept every new station while dropped the duplicate stations. The result shows that there are 524 stations been used recently. Then we transferred this station data into a geo data-frame.

The second one is crime data, we used all crime data staring from 2013 till now. For the crime data, we picked Primary Type, Longitude and Latitude column to make a list of tuple as location points and transferred them into a geo data-frame as we did with divvy data.

How we merge data

We set divvy points radian (300 meters) and join with crime data points, save the intermediate join data as join_data.csv, and used this for the website map and plot.

Mapping

After merging to get a complete data frame (“maps”) folium is imported for actual mapping. A “for” loop is used for each row in the “maps” data frame, and the html and Iframe functions in folium are used to create popup labels on markers, each of which indicates a bike station. The popup labels provide information on station name, station ID, and the number of crimes that have occurred in a 300 meter radius around that station in the past three years.

Plot Crime Type around the station

In our website, under "Crime Data Analysis" tab, you can find the plot of crimes near any station you choose under the dropdown menu. The crime plot has information of types of crimes occurred near the station you chose, also the number of crimes.

Using this information, you can avoid areas that are more dangerous when you plan your trip.
